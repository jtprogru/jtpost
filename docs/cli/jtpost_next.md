## jtpost next

Рекомендация следующего поста

### Synopsis

Рекомендует следующий пост для работы на основе дедлайнов, scheduled_at и статуса.

```
jtpost next [flags]
```

### Options

```
  -f, --format string   формат вывода (full, plain, json) (default "full")
  -h, --help            help for next
```
