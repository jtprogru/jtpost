# readme-en — английский перевод README

## Goal
Закрыть последний код-релевантный пункт этапа 13: English-перевод README.

## Scope
- Создать `README.en.md` рядом с существующим `README.md`.
- Полный перевод структуры и содержания.
- В начале обоих README — взаимные ссылки (Russian | English) для языкового переключения.
- Команды/код-блоки/имена флагов остаются как есть; переводим только описания.
- Не дублировать badges (они визуальные, языконезависимы).

## Out of scope
- Перевод docs/* (уже большой объём, отдельный cut если понадобится).
- Видео-туториалы.
- Переводы других md-файлов (CHANGELOG, ROADMAP).
